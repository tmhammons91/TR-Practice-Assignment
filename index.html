Software Defined Storage Overview


What is Software Defined Storage (SDS)?

Software defined storage (SDS) is defined at the most basic level as a type of storage solution where software is used to manage and control data storage independently from the underlying hardware. SDS solutions abstract the physical aspect of storage, thus allowing systems to be built using industry-standard hardware components. The storage environment is provisioned and managed programmatically through software APIs rather than hardware-based controllers.

Software defined storage is just one element in a trend toward “software defined everything.” Related concepts and product categories include software defined networks, software defined computing, software defined architecture, and software defined data centers. While definitions of these vary and often overlap, they are all part of an ongoing decoupling of software development from the physical infrastructure. The benefits of this trend in general and of software defined storage solutions in particular include flexibility in hardware choice, cost efficiencies in being able to use commodity components, ease of scalability, elasticity, and dynamic provisioning.


Software Defined Storage Features & Capabilities

  • Ability to use with commodity, industry-standard hardware: SDS solutions are not tied to proprietary hardware products or to one specific hardware vendor
  • API integrated: storage is dynamically configured and managed through software interfaces
  • Supports dynamic resource utilization: SDS solutions allow for resource utilization to be matched to resource needs; storage can grow and shrink as needed
  • Support for a full range of storage services: allows for compression, data deduplication, and virtualization, and supports block, file, and object storage interfaces
  • Utility-based billing: Enables a model where users pay only for what they need and what they use
  • Suitability for very large storage environments: SDS solutions are effective for dynamic, multi-tenant environments

Software Defined Storage Products


Product Offerings From Leading Storage Vendors and Manufacturers

  • Dell EMC: ScaleIO
  • NetApp: ONTAP Select

    ONTAP Select is a software defined storage solution from NetApp, one of the largest vendors in the data center market. ONTAP Select is based on NetApp’s original ONTAP software. It can be deployed on a range of commodity hardware components or used on the existing server infrastructure within an organization’s own data center. It promises the agility of cloud storage in that it can be quickly and easily deployed for remote locations or proof-of-concept project environments, but with the security and control of traditional on-premises storage. Licensing is done on a per-node/per-virtual machine basis.

  • HPE (Hewlett Packard Enterprise): StoreVirtual
  • Hitachi: VSP (Virtual Storage Platform)
  • IBM: Spectrum Storage Suite

Research/Content Strategy and Sources


Strategy

I chose this category for a very practical reason—I have a personal contact who has many years of experience with storage area networks and data centers and who was willing to provide an informal research interview on the topic. I began my research with this interview and continued it with various articles and websites. For the top 5 products, I chose to define “top” as being from the leading/largest vendors in the storage manufacturing space since some sources highlighted the fact that most of the leading vendors have jumped on the SDS bandwagon over the last several years. (From my research, I'm certain that there are several other ways the "tops" could be defined, such as top start-up vendors, top open-source SDS vendors, etc.) I followed up with my interviewee to get the names of the leading/largest storage vendors, and I then searched out the basic information on their SDS offerings.
